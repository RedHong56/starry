markdown# 🌟 Starry — AI 타로 점괘 모바일 애플리케이션

> 3D 타로술사 캐릭터와 AI 해석 엔진이 결합된 모바일 타로 앱

## 📖 프로젝트 소개

Starry는 사용자의 고민을 입력받아 과거-현재-미래 3장의 타로카드를 
기반으로 AI가 개인 맞춤형 점괘를 제공하는 모바일 애플리케이션입니다.

## 🛠 기술 스택

| 영역 | 기술 |
|---|---|
| 클라이언트 | Unity 2022.3 LTS, C# |
| 백엔드 | FastAPI, Python |
| AI | Google Gemini 2.0 Flash |
| 인프라 | Firebase, Railway |
| 광고/결제 | Google AdMob, Unity IAP |

## 📁 프로젝트 구조

\`\`\`
starry/
├── card_pipeline/    # AI 카드 자동 생성 파이프라인 및 이미지 합성
├── backend/     # FastAPI 서버
├── client/      # Unity 모바일 앱
├── assets/      # 3D 에셋 원본
└── docs/        # 기획 및 기술 문서
\`\`\`

## 🚀 주요 성과

- 🎴 78장 타로카드를 AI 자동 파이프라인으로 생성 (수작업 8시간 → 30분)
- 🤖 Gemini Flash 기반 개인 맞춤형 타로 해석 (응답 시간 평균 3초)
- 💰 Rewarded Ad + IAP 하이브리드 수익 모델
- 📱 iOS / Android 동시 지원
